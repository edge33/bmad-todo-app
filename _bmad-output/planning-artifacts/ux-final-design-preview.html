Design Specifications

Active Task State

Background: White (#FFFFFF)

Text: Warm Gray (#4a4a4a)

Icon: Emoji (📝 📞 📄 etc.)

Padding: 14px

Border Radius: 10px

Completed Task State

Background: White (#FFFFFF)

Left Border: 4px solid Green (#22c55e)

Text: Dark Green (#2d5a3d)

Icon: ✅ Checkmark

Font Weight: 500 (Medium)

Interaction States

Hover: Slight right translation + shadow

Active/Click: Bounce animation 300-400ms

Completion: Slide left + color change + emoji

Focus: Indigo ring for keyboard nav

Animation & Feedback

Completion Bounce

Duration: 300-400ms

Easing: cubic-bezier(0.68, -0.55, 0.265, 1.55)

Effect: Scale 1 → 1.04 → 1.01

Purpose: Draw attention, celebrate completion

Color Transition

Duration: 300ms

Easing: ease-out

Effect: Smooth gradient transition

Signal: "Your task is done"

Movement Animation

Duration: 350ms

Easing: ease-out

Effect: Slide down to Completed section

Signal: Task organized and archived

Interaction Feedback

Hover: 0.2s transition

Transform: translateX(4px)

Shadow: Depth enhancement

Purpose: Invite interaction

Accessibility & Responsive

Contrast Ratios

✓ Text on white: 7.8:1 (WCAG AAA)

✓ Active vs Completed: Clear visual distinction

✓ Multiple signals: Color + icon + border + position

✓ Color-blind friendly: Checkmark + border

Touch & Keyboard

✓ Touch targets: 44x44px minimum

✓ Keyboard nav: Tab, Enter, Delete keys

✓ Focus states: Clear visual indicator

✓ Enter submits, Delete removes

Responsive Design
